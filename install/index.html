インストール

Yapps がご利用の環境にインストールできるようになりました！

Yapps は Web ブラウザ上で動作する Web アプリケーションですが、 PWA (Progressive Web Apps) に対応しているブラウザであれば、Yapps をPCのデスクトップ画面やスマホのホーム画面に追加して、すぐに Yapps にアクセスできるようにすることができます。
また、インストールにより各種ツールをオフラインで利用することも可能です（一部除く）。

PWA (Progressive Web Apps) とは？

PWA は、Web アプリケーションをネイティブアプリケーションのように振る舞わせる技術です。
PWA に対応しているブラウザであれば、Web アプリケーションをインストールして、オフラインで利用することができます。
実際には Web ブラウザ上で動作しますので、引き続き Web ブラウザと同じようにご利用いただけます。

PWA についての詳細は MDN Web Docs をご覧ください。
https://developer.mozilla.org/ja/docs/Web/Progressive_web_apps

PWA のインストール方法

以下の手順で、お使いのブラウザごとに Yapps をインストールできます。

PC

Google Chrome

  1. Yapps を Chrome で開きます。
  2. アドレスバーの右側にある「インストール」アイコンをクリック、またはメニュー（︙）を開き「Yapps をインストール」を選択します。
  3. 「インストール」をクリックすると、Yapps がアプリとして登録されます。

Microsoft Edge

  1. Yapps を Edge で開きます。
  2. アドレスバーの右側にある「アプリのインストール」アイコン（パソコンのマーク）をクリックします。
  3. 表示されるポップアップで「インストール」をクリックします。

Safari, Firefox（非対応）

現在、Safari および Firefox はデスクトップ版の PWA インストールをサポートしていません。

モバイル

Android（Google Chrome, Microsoft Edge, Firefox）

  1. Yapps を対応ブラウザで開きます。
  2. 各ブラウザでホーム画面に追加します。
    • Google Chrome / Edge の場合：メニュー（︙）を開き「ホーム画面に追加」を選択します。
    • Firefox の場合：「ホーム画面に追加」機能を使用してショートカットを作成できます（PWAとしての機能は制限されます）。
  3. 「追加」をタップすると、ホーム画面にアイコンが作成されます。

iOS（Safari）

  1. Yapps を Safari で開きます。
  2. 共有メニュー（四角と上向き矢印のアイコン）を開き「ホーム画面に追加」を選択します。
  3. 「追加」をタップすると、ホーム画面にアイコンが作成されます。
